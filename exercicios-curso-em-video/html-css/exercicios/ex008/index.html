Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em itálico usando a tag EM (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado com a tag del.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag ins (semântica).

Existe também o sublinhado com a tag U (não semântica).

Texto Sobrescrito

Para inserir coisa do tipo x20+3. Isso é feito com a tag sup.

Texto Subscrito

Para inserir coisa do tipo H2.O. Isso é feito usando a tag sub.
